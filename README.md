# ram_opz_mapped

> OP-Z-mapped generative sequencer with euclidean rhythms

## Controls

- **E2** — select track
- **E3** — adjust density
- **K2** — start / stop
- **K3** — regenerate track

## Grid

None required.

## Requirements

- norns
- MIDI device (OP-Z or compatible)

## Install

```
;install https://github.com/jamminstein/ram_opz_mapped
```
